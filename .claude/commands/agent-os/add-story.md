# Add Story

Add a new user story to an existing specification.

Refer to the instructions located in agent-os/workflows/core/add-story.md

**Use Case:**
- Neue Anforderung während laufender Implementierung
- Erweiterung einer bestehenden Spec
- Nachträgliche Feature-Ergänzung

**Workflow:**
- Spec auswählen (oder als Parameter angeben)
- PO gathers fachliche requirements (kurzer Dialog)
- Architect adds technical refinement (WAS/WIE/WO/WER)
- Story-ID folgt Spec-Präfix (z.B. PROF-004)
- Story wird zu stories/ und story-index.md hinzugefügt
- Falls Kanban existiert: Story kommt ins Backlog
